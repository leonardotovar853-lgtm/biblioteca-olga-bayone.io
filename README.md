# biblioteca-olga-bayone.io
Plataforma web de biblioteca digital y repositorio institucional para la U.E. Olga Bayone de Rodríguez. Diseñada con arquitectura híbrida: frontend veloz (HTML5/CSS3/JS) y backend dinámico en Python con Flask. Cuenta con autenticación segura Google Sign-In (@olgabayone.com), base de datos en Google Sheets y buscador con filtros avanzados.
