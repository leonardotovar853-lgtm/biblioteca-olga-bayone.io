¿Cómo registrar un recurso?

Elige el tipo de recurso, genera un enlace compartido y completa el formulario. En pocos minutos tu material estará listo para uso escolar.

Paso 1

Sube tu archivo a Google Drive o a la plataforma de almacenamiento que uses y copia el enlace de uso compartido.

Paso 2

Asegúrate de que el enlace sea público o tenga permisos de acceso para cualquier persona con el enlace.

Paso 3

Completa el formulario de registro con los datos del recurso, descripción y el enlace compartido.
